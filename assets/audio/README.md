# Darija30 Audio Folder

Place real Moroccan Darija MP3 files here.

Required folders:

```text
day01/
day02/
day03/
```

The exact file list is generated from `js/data/lessons-data.js` and stored in:

```text
content/AUDIO_MANIFEST.csv
content/AUDIO_MANIFEST.md
```

Important: do not add music or sound effects. Darija30 is built around clear speech only.

If an audio file is missing, the website now shows a friendly “Recording needed” message instead of failing silently.
