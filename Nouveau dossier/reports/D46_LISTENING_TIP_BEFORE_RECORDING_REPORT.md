# D46 — Listening Tip Before Recording

## هدف المرحلة
إضافة توصية خفيفة داخل صفحة الجملة تشجع المتعلم على الاستماع المتكرر قبل تسجيل صوته.

## ما تم
- أضيف سطر نصي صغير بعد أزرار الاستماع وقبل قسم التسجيل.
- النص المعتمد للواجهة الإنجليزية:
  `🎧 Listen a few times. Then say it like you’re really in Morocco.`
- أضيف تنسيق CSS خفيف بدون كرت كبير حتى لا تتحول النصيحة إلى شرح طويل.

## ما لم يتم تغييره
- لم يتم تغيير الصور.
- لم يتم تغيير الفيديوهات.
- لم يتم تغيير صفحة اختبار الذاكرة.
- لم يتم تغيير منطق التسجيل أو الصوت.
