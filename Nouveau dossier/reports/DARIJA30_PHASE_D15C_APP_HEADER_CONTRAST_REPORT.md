# Darija30 Phase D15C — App Header Contrast Fix

## الهدف
إصلاح ظهور روابط هيدر التطبيق بلون أبيض غير واضح على صفحات التطبيق مثل `#/app/lessons` و `#/app/lesson/16`.

## ما تم
- تغيير هيدر التطبيق من خلفية داكنة/نص أبيض إلى خلفية بيضاء واضحة مع نص أسود.
- جعل الرابط النشط يظهر بلون Terracotta أحمر مغربي مع خط سفلي صغير.
- جعل الهيدر مثبتاً أعلى صفحة التطبيق بدون فراغ `top-16` بعد إخفاء الهيدر العام.
- إضافة CSS حماية حتى تبقى روابط التطبيق واضحة دائماً.

## الملفات المعدلة
- `index.html`
- `css/styles.css`

## لم يتم تغييره
- لا تعديل على الدروس.
- لا تعديل على الصوت أو الفيديو.
- لا تعديل على Backend أو Stripe.
