# D28 Lesson Polish Model

## Lesson page model
Each lesson still contains 5 phrase pages.

The learner sees:
- Scenario
- Goal
- Video / dialogue when available
- Core phrase
- Audio buttons
- Meaning / cultural intent
- Moment Visual when the phrase has an image
- Culture note
- Memory hook
- Phrase navigation
- Quick Check

## New UI behavior
- A phrase stepper shows Phrase 1–5 clearly.
- Quick Check is no longer permanently open.
- The learner can open or hide Quick Check.
- The chosen state is saved in the browser.

## Product reason
The lesson page stays rich, but the bottom of the page is cleaner and less heavy, especially on desktop and mobile.
