# Darija30 Real Audio QA Checklist — Phase D12

Use this checklist after importing the MP3 files.

## Technical checks
- [ ] All 30 MP3 files exist in `assets/audio/day01`, `day02`, and `day03`.
- [ ] `#/audio-qa` shows `30/30 ready`.
- [ ] No file path contains spaces or uppercase surprises.
- [ ] Every audio button works on Chrome desktop.
- [ ] Every audio button works on mobile browser.
- [ ] Normal and slow files are not accidentally swapped.

## Content checks
- [ ] Voice is Moroccan Darija, not MSA.
- [ ] Pronunciation matches the Friendly Latin line.
- [ ] Normal speed sounds natural.
- [ ] Slow speed is clear but not robotic.
- [ ] No background noise.
- [ ] No music.
- [ ] No echo.
- [ ] Volume is consistent across all files.

## Product checks
- [ ] Audio supports the scenario-first method.
- [ ] Learner can repeat the phrase after listening once or twice.
- [ ] Cultural explanation still matches the recorded phrase.
- [ ] Missing-audio warning no longer appears in the free lesson.
