# Darija30 Transparent Memory Bubble Method

The Moment Visual should remain visible. The Memory Bubble is only a cue for the phrase, not the main visual object.

## Rule
- Image = visual context.
- Bubble = phrase memory cue.
- Video = full dialogue.

## Visual system
- Bubble fill: semi-transparent white.
- Bubble border: dark and readable.
- Phrase text: dark navy/black.
- Bubble tail: same semi-transparent fill.
- Light blur is acceptable to preserve readability.

## Reason
A solid white bubble hides the scene. A transparent bubble keeps the phrase readable while preserving the visual story behind it.
