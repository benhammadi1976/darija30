# D30 Homepage Positioning Model

## Product identity
Darija30 is a Morocco Survival Speaking Trainer.

It is not positioned as:
- a full academic Arabic course,
- a grammar-first Darija course,
- a dictionary,
- a passive phrasebook.

## Core promise
30 days.
30 lessons.
150 useful Moroccan Darija phrases.
Real Morocco situations.
No Arabic alphabet required.

## User target
English-speaking travelers, visitors, and beginners who need useful Moroccan Darija quickly.

## Homepage message
The homepage should make the user understand in a few seconds:

1. This is for real Morocco travel moments.
2. It teaches speaking, not just reading.
3. It starts with sound and situations.
4. Friendly Latin Darija is primary.
5. Arabic script is optional.
6. Level 1 is a clear 30-day path.

## Funnel
First 3 lessons are free:
- Taxi Survival
- Café Culture
- Market Price

After the free lessons, the user can unlock Level 1.
