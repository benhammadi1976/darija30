# Darija30 Admin Role + Lock Preview

This static version has two visual roles:

## Learner / Visitor
- Opens free lessons fully.
- Sees locked paid lessons as preview only.
- Does not see full paid lesson content.

## Admin Preview
- Activated from `/admin` using the demo button.
- Sees the admin content map.
- Can open locked lessons in full to review content, paths, audio/video needs, and images.
- The lesson still displays that the lock is active for learners.
- Can click “View as learner” to see the protected preview exactly like a student.

## Logout
After admin logout, localStorage admin preview mode is cleared and the site behaves again as a learner/visitor website.

This phase is for product visualization only before real Backend, database, upload system, and admin authentication.
