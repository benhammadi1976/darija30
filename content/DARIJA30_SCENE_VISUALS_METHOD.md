# Darija30 Scene Visuals Method — Phase D21

## Purpose
Scene Visuals are small contextual illustrations inside phrase lesson pages. They are not a second video and not a text explanation.

They create a quick memory image for the phrase:
- Taxi: traveler asking a driver to go to a hotel.
- Café: traveler ordering black coffee.
- Market: traveler asking the price of an item.

## Learning role
Each phrase page now has three layers:
1. Core phrase text and audio.
2. Video dialogue showing what the traveler hears and says.
3. Moment Visual as a still image memory cue.

## Rules
- No text inside the illustration.
- Same warm premium illustration style.
- Keep it clear at small size.
- Use only for phrases with a strong visual situation.
- Do not replace video with the illustration.
