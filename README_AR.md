# D69-FIX-9 — دعوة اختيارية لعجلة الأسبوع بعد الاختبار اليومي

## ما تم
- لم يعد Day 7 / 14 / 21 / 28 يفتح عجلة الأسبوع بدل الاختبار اليومي.
- زر **Do you remember?** في أيام checkpoint يفتح الاختبار اليومي العادي أولًا.
- بعد إكمال آخر موقف في الاختبار اليومي لليوم الأسبوعي، ينتقل المتعلم تلقائيًا إلى صفحة دعوة اختيارية.
- تظهر بطاقة:
  - **حان وقت الاختبار الأسبوعي إن رغبت**
  - زر **ابدأ الاختبار الأسبوعي**
  - زر **تجاوز الآن إلى اليوم التالي**
- عند الضغط على **ابدأ الاختبار الأسبوعي** تفتح عجلة الأسبوع كما هي.
- عند الضغط على **تجاوز الآن** ينتقل إلى اليوم التالي.

## لم يتم تغييره
- الاختبار اليومي نفسه.
- منطق العجلة والدوران.
- Supabase.
- الأدمن.
- رفع الملفات.
