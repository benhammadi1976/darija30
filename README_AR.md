# D70-FIX-2 — Progressive Memory Challenge Levels

## ما تم
- إضافة تجربة مستويات الذاكرة التدريجية لعجلة المراجعة الأسبوعية.
- Week 1 يعرض Memory Level 1 فقط: 5 / 10 / 15 / 20 / 25 / 30 / 35.
- Week 2 يعرض Level 1 + Level 2: حتى 70 موقفًا.
- Week 3 يعرض Level 1 + Level 2 + Level 3: حتى 105 موقفًا.
- Week 4 يعرض Level 1 + Level 2 + Level 3 + Level 4: حتى 140 موقفًا.
- الأرقام تعني عدد المواقف التي يريد المتعلم الإجابة عنها، وليست أرقام نتيجة العجلة.
- عند hover على أي رقم تظهر بطاقة شرح.
- تقليل ازدحام أرقام العجلة عند البنوك الكبيرة بعرض علامات مختصرة فقط.

## الملفات المعدلة
- `index.html`
- `js/weekly-memory-levels-fix2.js`

## ملاحظات تقنية
- هذا الإصلاح يحمّل نسخة runtime patched من `js/lesson-player.js` ثم يعيد تفعيل `window.DarijaLessonPlayer` بنفس التدفق الحالي.
- لم يتم تغيير الاختبار اليومي أو منطق الضغط على العجلة.
