# D69-FIX-10 — Transparent English Weekly Prompt Overlay

## ما تم
- تحويل دعوة الاختبار الأسبوعي بعد الاختبار اليومي إلى بطاقة شفافة فوق صفحة العجلة.
- تغيير النص إلى الإنجليزية.
- إزالة شكل الصفحة البيضاء الكبيرة المستقلة.
- إبقاء العجلة وخلفية الصور الذهنية ظاهرة خلف البطاقة.
- زر `Start Weekly Wheel` يفتح تجربة العجلة.
- زر `Skip to Day X` يتجاوز الاختبار الأسبوعي إلى اليوم التالي.

## لم يتغير
- الاختبار اليومي نفسه.
- دوران العجلة.
- Supabase.
- الأدمن.
- رفع الملفات.

## الملفات المعدلة
- `js/lesson-player.js`
- `css/styles.css`
