# Darija30 — النسخة د57

## ماذا تم؟
تم حذف ظهور قسم **Bonus Phrases** من صفحة الدرس/الجملة.

## لماذا؟
حتى تبقى تجربة المتعلم المبتدئ بسيطة:
- استماع
- تسجيل
- I’m in Morocco
- Learned
- Do you remember?

## ما لم يتغير
- الجمل الخمس الأساسية كما هي.
- الفيديوهات كما هي.
- التسجيل كما هو.
- صفحة Do you remember? كما هي.
- بيانات Bonus Phrases بقيت محفوظة في البيانات للمستقبل، لكنها لا تظهر للمتعلم الآن.

## تقرير المرحلة
`reports/D57_REMOVE_BONUS_PHRASES_REPORT.md`


## D68B — تنظيف لوحة الأدمن وتجهيز 12 مستوى مستقل

- تم جعل **مركز ملفات الدروس** هو المكان الرئيسي لإدارة الجمل والملفات.
- تم تنظيف شريط الأدمن من روابط مكررة مثل إدارة الدروس وإدارة الجمل.
- تم إضافة اختيار **Level 1 إلى Level 12** داخل مركز الملفات.
- كل مستوى مصمم كمجال مستقل: Level → Day/Lesson → Phrase.
- مستوى Level 1 يستعمل الدروس الحالية كما هي، والمستويات 2–12 تظهر كمساحات مستقلة جاهزة لاحقاً بدون إعادة بناء اللوحة.
- أبقينا الرفع والتعديل الديناميكي في Supabase كما كان في D68/D68C.
- لم يتم تغيير صفحة المتعلم أو مسارات الصوت/الفيديو الحالية الموجودة فعلاً.


## D68B-FIX — حذف صفحات الأدمن القديمة

- تم حذف صفحة `/admin/lessons` القديمة من `index.html` والراوتر.
- تم حذف صفحة `/admin/phrases` القديمة من `index.html` والراوتر.
- تم حذف رابط Admin lessons القديم من معاينة الدرس واستبداله بمركز ملفات الدروس.
- لم يعد هناك كارت “هذه الصفحة أصبحت مدمجة”. مركز ملفات الدروس هو المكان الرسمي الوحيد لإدارة الدروس والجمل والملفات.
- عند فتح المسارات القديمة يدويًا يتم تحويل المستخدم إلى `#/admin/lesson-media` لحماية التجربة من الروابط القديمة.

## D68D-E — بوابة الأدمن وتسمية مركز ملفات الدروس

- تمت إضافة صفحة دخول أدمن مستقلة: `#/admin/login`.
- صفحة المتعلم `#/login` بقيت للمتعلم فقط ولم يتم خلطها مع دخول الأدمن.
- أصبحت بوابة الأدمن تستعمل واجهة بسيطة: Admin Email + Admin Code.
- Admin Code هو كلمة سر Supabase الآمنة، وليس كوداً ثابتاً مكشوفاً داخل الموقع.
- تم نقل المسار الرسمي لمركز ملفات الدروس من `#/admin/audio` إلى `#/admin/lesson-media`.
- الروابط القديمة `#/admin/audio` و `#/admin/phrases` و `#/admin/lessons` تتحول تلقائياً إلى `#/admin/lesson-media`.
- تم تنظيف لوحة مركز الملفات من حقول Project URL و Publishable Key اليومية؛ بقيت فقط في إعداد تقني متقدم داخل بوابة الأدمن عند الحاجة لأول مرة.
- بعد الدخول الناجح يفتح الموقع مباشرة مركز ملفات الدروس.
- تم تحديث صفحة Reset Password لتعود إلى بوابة الأدمن بدلاً من صفحة المتعلم.
- فحص JavaScript: `node --check` ناجح لكل ملفات `js`.

## D68DE-FIX — ترتيب مركز ملفات الدروس وإصلاح تداخل الشريط

- تم ترتيب صفحة `#/admin/lesson-media` بالترتيب الرسمي: العنوان، الإحصائيات، بوابة الأدمن، قاعدة تسمية الملفات، اختيار المستوى/الدرس، ثم جدول الدرس المختار.
- تم إنزال محتوى الصفحة أسفل شريط الأدمن الأزرق حتى تبقى الأزرار وحالة الدخول ظاهرة.
- لم يتم تغيير Supabase أو نظام الرفع أو صفحة المتعلم.
- تم تحديث cache query لملفي `styles.css` و `admin-content-manager.js`.
